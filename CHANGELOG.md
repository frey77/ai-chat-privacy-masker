# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-07-07

### Added

- Initial Chrome / Edge extension popup for masking JSON and plain-text error content
- Default full masking with configurable prefix and suffix preservation
- Tree-based field review with per-node and per-field masking controls
- Searchable field tree with sensitive-key highlighting
- Local project documentation, license, and GitHub-ready repository structure
- Extension icons and README screenshot assets
